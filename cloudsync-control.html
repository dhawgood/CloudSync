System Setup Required

1. Install rclone

rclone is required for cloud synchronization

macOS: brew install rclone

Linux: curl https://rclone.org/install.sh | sudo bash

Windows: Download from https://rclone.org/downloads/

2. Configure Google Drive

Set up Google Drive connection with name "gdrive-rclone"

Run: rclone config

Choose "New remote" → "Google Drive" → Name: "gdrive-rclone"

3. Verify Permissions

Ensure CloudSync can access your Documents folder

Check that ~/Documents exists and is readable
